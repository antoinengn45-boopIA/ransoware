# ⚠ Security Alert System (UI Simulation)

## 📌 Description
This project is a **cybersecurity UI simulation** inspired by ransomware-style interfaces.  
It is strictly educational and does not encrypt, modify, or access any files.

## 🎯 Purpose
- Teach cybersecurity awareness
- Demonstrate ransomware UI patterns
- Show how social engineering interfaces look
- Practice HTML/CSS/JavaScript structure

## 🚀 Features
- Fake encryption countdown timer
- "System compromised" interface
- Decryption key input system
- Dynamic status feedback
- Dark hacker-style UI
